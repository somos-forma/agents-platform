## Overview

Una aplicación web moderna para orquestar agentes de AI, construida con las últimas tecnologías web.

### Tecnologías Utilizadas

- **Next.js 15.5.3** - Framework de React con App Router
- **React 19.1.0** - Biblioteca de interfaz de usuario con las últimas características
- **TypeScript 5.6** - Tipado estático para JavaScript
- **Tailwind CSS v4** - Framework de CSS utility-first
- **shadcn/ui** - Componentes de UI modernos y accesibles

### Características de React 19

Esta aplicación utiliza React 19.1.0, que incluye:
- **React Compiler** - Optimizaciones automáticas de rendimiento
- **Actions** - Manejo simplificado de formularios y mutaciones
- **use() Hook** - Nueva forma de consumir promesas y contexto
- **Mejoras en Suspense** - Mejor manejo de estados de carga
- **Nuevas APIs de metadatos** - Mejor SEO y gestión de metadatos

### Notas de Compatibilidad

- React 19 está completamente soportado por Next.js 15.5.3
- shadcn/ui tiene soporte completo para React 19 en su versión canary
- Todas las dependencias de Radix UI son compatibles con React 19
- Los tipos de TypeScript han sido actualizados a las versiones compatibles con React 19

## Instalación Local

\`\`\`bash
# Instalar dependencias
npm install

# Si encuentras problemas con peer dependencies, usa:
npm install --legacy-peer-deps

# Ejecutar en modo desarrollo
npm run dev

# Construir para producción
npm run build
\`\`\`

### Notas de Instalación

- Al usar npm, es posible que necesites usar `--legacy-peer-deps` debido a las actualizaciones de React 19
- PNPM, Bun y Yarn no requieren flags adicionales
- Todas las dependencias han sido verificadas para compatibilidad con React 19
